Cloud Security Tools

Monitor cloud posture across AWS, Azure, and GCP. Detect cloud threats with CDR. Scan container images for vulnerabilities.

get_cloud_risk

Cloud security posture — CIS benchmark failures, misconfigurations, account-level risk across providers. Also serves as CDR findings viewer when include_threats=True.

Parameters

NameTypeDefaultDescription
include_threatsboolFalseInclude CDR threat detections alongside posture data
daysint30Lookback window in days
severitystr""Filter by severity (e.g., CRITICAL, HIGH)
cloud_providerstr""Filter by cloud provider (AWS, AZURE, GCP)

Examples

You askTool call
"What's our cloud security posture?"get_cloud_risk()
"How many CIS controls are failing in AWS?"get_cloud_risk()
"What cloud threats were detected this week?"get_cloud_risk(include_threats=True, days=7)
"Show me critical CDR findings in AWS"get_cloud_risk(include_threats=True, severity="CRITICAL", cloud_provider="AWS")
"Are there any crypto-miners in our cloud?"get_cloud_risk(include_threats=True, days=30)
"Show me Azure threat detections from last month"get_cloud_risk(include_threats=True, days=30, cloud_provider="AZURE")

get_cdr_findings

Cloud Detection & Response findings from TotalCloud. Returns threat detections, suspicious activities, and security incidents across cloud environments.

Parameters

NameTypeDefaultDescription
daysint7Lookback window in days
severitystr""Filter by severity (e.g., CRITICAL, HIGH)
cloud_providerstr""Filter by cloud provider (AWS, AZURE, GCP)

Examples

You askTool call
"What cloud threats were detected recently?"get_cdr_findings()
"Show me critical CDR findings from the last 3 days"get_cdr_findings(days=3, severity="CRITICAL")
"Any GCP security incidents this week?"get_cdr_findings(days=7, cloud_provider="GCP")

get_image_vulns

Container image vulnerability scanning. Returns vulnerabilities found in a specific container image, including severity, CVEs, and fix availability.

Parameters

NameTypeDefaultDescription
image_idstrrequiredThe container image ID or digest to scan
limitint50Maximum number of vulnerabilities to return

Examples

You askTool call
"What vulns are in container image sha256:abc123?"get_image_vulns(image_id="sha256:abc123")
"Show me the top 10 vulns in image img-456"get_image_vulns(image_id="img-456", limit=10)
