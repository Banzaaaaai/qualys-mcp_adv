Compliance & Remediation Tools

Policy compliance monitoring, remediation ticket management, SLA tracking, and vulnerability exception management.

get_compliance_posture

Policy compliance from the Qualys PC module. Returns pass/fail rates by framework (CIS, PCI-DSS, HIPAA, NIST, SOC2) and top failing controls.

Parameters

NameTypeDefaultDescription
frameworkstr""Filter by compliance framework (CIS, PCI-DSS, HIPAA, NIST, SOC2)
platformstr""Filter by platform (Linux, Windows, etc.)
limitint20Maximum number of failing controls to return

Return Shape

{
  "framework": "CIS",
  "total_controls": 245,
  "passed": 198,
  "failed": 47,
  "pass_rate": 80.8,
  "top_failing_controls": [
    {
      "control_id": "4.1.1",
      "title": "Ensure auditing is enabled",
      "failing_assets": 23,
      "severity": "HIGH"
    }
  ]
}

Examples

You askTool call
"What's our CIS Benchmark compliance score?"get_compliance_posture(framework="CIS")
"Show me failing PCI-DSS controls"get_compliance_posture(framework="PCI-DSS")
"What's our Linux policy compliance rate?"get_compliance_posture(platform="Linux")

get_remediation_tickets

Remediation ticket tracking. Filter by status, assignee, or overdue flag to manage your remediation workflow.

Parameters

NameTypeDefaultDescription
statusstr"OPEN"Filter by ticket status (OPEN, CLOSED, IN_PROGRESS)
assigneestr""Filter by assigned user
overdueboolFalseShow only overdue tickets

Examples

You askTool call
"What remediation tickets are open?"get_remediation_tickets(status="OPEN")
"Show me overdue remediation tickets"get_remediation_tickets(overdue=True)
"What tickets are assigned to jsmith?"get_remediation_tickets(assignee="jsmith")

create_remediation_ticket

Create a remediation ticket for a QID + asset combination. Links the vulnerability finding to an actionable ticket for tracking.

Parameters

NameTypeDefaultDescription
qidstrrequiredThe Qualys QID to remediate
asset_idstrrequiredThe asset ID where the vulnerability exists
assigneestr""User to assign the ticket to

Examples

You askTool call
"Create a ticket for QID 376267 on asset 123"create_remediation_ticket(qid="376267", asset_id="123", assignee="jsmith")

get_sla_status

SLA compliance — open/closed/overdue counts, compliance rate, and mean time to remediate (MTTR) by severity.

Examples

You askTool call
"What's our SLA compliance rate?"get_sla_status()
"What's our mean time to remediate?"get_sla_status()

get_vuln_exceptions

Vulnerability exception lifecycle — active waivers, false positives, and exceptions approaching expiry.

Parameters

NameTypeDefaultDescription
days_to_expiryint30Show exceptions expiring within this many days
vuln_typestr""Filter by exception type (e.g., false_positive)

Examples

You askTool call
"What exceptions expire soon?"get_vuln_exceptions(days_to_expiry=30)
"Show me all false positive exceptions"get_vuln_exceptions(vuln_type="false_positive")
