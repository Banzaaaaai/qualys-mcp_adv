Example Questions

Example security questions organized by workflow tool, showing how each question maps to the 7 tools in Qualys MCP v0.1.0.

Investigation Questions

1. Investigate CVE-2021-44228 (Log4Shell) completely investigate(target="CVE-2021-44228", depth="deep")
2. Are we exposed to Lazarus Group? investigate(target="Lazarus Group")
3. What's happening on 10.0.0.1? investigate(target="10.0.0.1")
4. What ransomware vulns exist? investigate(target="ransomware", threat_type="Ransomware")
5. What vulnerabilities affect Apache? investigate(target="Apache", software="Apache")
6. Show me EDR events this week investigate(target="endpoint threats", scope="edr", days=7)
7. File changes on production servers investigate(target="production", scope="fim", days=1)
8. CISA KEV vulnerabilities affecting us investigate(target="CISA KEV", threat_type="Cisa_Known_Exploited_Vulns")

Risk Assessment Questions

9. What's our overall risk? assess_risk()
10. Cloud risk in AWS assess_risk(scope="cloud", provider="aws")
11. Top risky assets assess_risk(scope="assets", sort_by="trurisk")
12. Container image vulnerabilities assess_risk(scope="containers")
13. Certificates expiring in 30 days assess_risk(scope="certs", days=30)
14. EOL systems in our environment assess_risk(scope="assets", eol_only=True)
15. Web app vulnerabilities for customer-portal assess_risk(scope="web", app_name="customer-portal")
16. Weak ciphers or TLS 1.0 usage assess_risk(scope="certs", weak_ciphers=True)

Compliance Questions

17. Are we PCI compliant? check_compliance(framework="PCI")
18. Failing CIS controls check_compliance(framework="CIS")
19. HIPAA compliance posture check_compliance(framework="HIPAA")
20. Risk acceptances expiring soon check_compliance(include_exceptions=True, days_to_expiry=30)

Remediation Questions

21. What should we patch first? plan_remediation()
22. Outstanding Windows patches plan_remediation(scope="patches", platform="windows")
23. Mitigation for CVE-2024-3400? plan_remediation(scope="mitigations", cves=["CVE-2024-3400"])
24. What's missing from our security program? plan_remediation(scope="program")

Security Overview Questions

25. Morning security briefing security_overview(period="today")
26. What happened this week? security_overview(period="week")
27. Quick environment snapshot security_overview(quick=True)
28. Scanner status security_overview(scope="infrastructure")

Report & Admin Questions

29. List available reports reports(action="list")
30. What's cached? cache_status()
